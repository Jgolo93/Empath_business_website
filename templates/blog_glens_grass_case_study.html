3 The Full Stack — Explained Simply

You don't need to be a tech expert to understand what we built. Here's the breakdown in plain English:

🌐

Website & Domain

We used a modern static site generator deployed to a global CDN. This means the site loads fast anywhere in South Africa (or the world). The .co.za domain was registered through a local registrar for R50/year.

📧

Email Setup

Zoho Mail's free tier provides professional email hosting. We configured DNS records (MX, SPF, DKIM) to ensure emails land in inboxes, not spam folders. The result: info@glensgrass.co.za works exactly like any corporate email.

📅

Booking System

A serverless database (Firebase) stores bookings. No server to maintain. No monthly hosting fees for database infrastructure. The booking form feeds directly into this database, available 24/7.

📱

SMS Automation

GitHub Actions (a free automation service) checks the database every hour. If an appointment is scheduled within the next few hours, it triggers an SMS via a low-cost SMS gateway. Customers get reminded automatically. Glen doesn't lift a finger.

💡
Pro Tip: The "serverless" approach means you only pay for what you use. No monthly server bills when you're just starting out. As your business grows, the system scales automatically.
